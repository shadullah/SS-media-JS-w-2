VAR:
• Function Scope: var is function-scoped. it is accessible within a function. • Hoisting: var variables are hoisted to the top of their scope and initialized with a value of undefined.
LET:
• Block Scope: let can be accessible within a block scope. • Hoisting: let variables are hoisted to the top of their scope but they are not initialized. if initialized then it will give a reference error.
CONST:
• Block Scope: Like let, const is also block-scoped. • Hoisting: like let, const declarations are hoisted to the top but are not initialized.
